# はじめかた（5分）

不動産の現場で使う AI アシスタント（Claude など）に、**業界用語の理解と、物確メモ・反響・
広告文の整理**を覚えさせるための小さな部品です。プログラミングの知識はいりません。

> 何ができる？ — 「この物確メモを整理して」「この反響の希望条件をまとめて」「"さんため"って何？」
> といったお願いを、AI が不動産の言葉のまま正しく処理できるようになります。

---

## 用意するもの

1. **MCP に対応した AI クライアント**（どれか1つ）
   - **Claude Desktop**（一番かんたん・アプリを入れるだけ）/ Claude Code / Cursor / Cline / OpenAI Codex など
2. **Node.js** または **Python 3**（多くのPCに既に入っています。無ければ Node.js を入れるのが簡単）

---

## 手順

### ① ワンコマンドで設定する

ターミナル（Mac は「ターミナル」、Windows は「PowerShell」）で次を実行します。
お使いのクライアントを選ぶと、設定を自動で書き込みます。

```bash
npx -y fudosan-ontology init
# Python 派の方は:  uvx fudosan-ontology init
```

> 自分で設定ファイルに書く場合（上級者）：AI クライアントの「MCP サーバ設定」に次の1行を足すだけ。
> ```json
> { "mcpServers": { "fudosan-ontology": { "command": "npx", "args": ["-y", "fudosan-ontology", "serve"] } } }
> ```

### ② AI クライアントを再起動する

設定を読み込ませるため、Claude Desktop などを一度閉じて開き直します。

### ③ 動いているか確認する

```bash
npx -y fudosan-ontology selftest
```
`✓ selftest pass` と出れば準備完了です。

## 最初に試す（実際の仕事で）

AI に**ふつうの日本語で**お願いするだけ。下を貼って試してみてください。

**1. 物確メモを整える**
> この物件メモを正規化して：「中野区中央三丁目12番5号、2LDK、専有48.6㎡、平成20年築、南東向き、ペット相談可」

→ 住所が `中野区中央3-12-5`、面積 `48.6㎡（約14.7坪）`、築年 `2008年築（築18年）`、方位 `南東`、
   条件 `ペット可` に整理されます。

**2. 反響メールから希望条件を拾う**
> このお客様の希望をまとめて：「中野区か杉並区で2LDKを探してます。家賃15万円以内、駅徒歩10分以内、猫飼ってます」

→ エリア（中野区・杉並区）/ 間取り(2LDK) / 賃料上限(15万円) / 駅徒歩(10分以内) / ペット可 に構造化されます。

**3. 業界用語をその場で引く**
> 「さんため」「あんこ」「マイソク」ってどういう意味？

→ 正規形・やさしい説明・カテゴリで返ってきます。

## 困ったとき

- **AI がツールを使ってくれない** → クライアントを再起動。設定が読み込まれていない可能性。
- **`npx`/`uvx` が無いと言われる** → Node.js（npx 用）か、uv / Python（uvx 用）を入れてください。
- **設定ファイルの場所** → Claude Desktop は `~/Library/Application Support/Claude/claude_desktop_config.json`（Mac）。
  `npx -y fudosan-ontology config claude-desktop` で場所と設定例を表示できます。
