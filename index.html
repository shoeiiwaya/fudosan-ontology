AI から使う（MCP・インストール不要）

お使いの AI クライアントの設定に 1 行。Claude Code / Claude Desktop / OpenAI Codex / Cline / Cursor などで動きます。

npx（Node）
{ "mcpServers": { "fudosan-ontology": { "command": "npx", "args": ["-y","fudosan-ontology","serve"] } } }
uvx（Python・全機能）
{ "mcpServers": { "fudosan-ontology": { "command": "uvx", "args": ["fudosan-ontology","serve"] } } }

クライアントを選んで自動設定する導入ウィザード。ローカル LLM を選ぶと搭載メモリを見て推奨モデルを提示します。

npx -y fudosan-ontology init # または: uvx fudosan-ontology init npx -y fudosan-ontology selftest # 動作確認

できること

📖 名寄せ辞書

377語/10ドメイン・2,400+エイリアス。漢字/カナ/英字・全角半角・送り仮名・略語の表記ゆれを正規形へ。業者向け定義／顧客向けやさしい定義／英訳つき。

🔤 正規化ハーネス

住所・希望条件・面積/築年/方位・物件条件タグを標準形に。可否や法令は確定せず「要確認」に倒す安全設計。

🤖 MCP サーバ

resolve_term / normalize / assess_risk。改行区切り JSON-RPC・依存ゼロ・ネットワーク呼出なし。AI からそのまま呼べる。

🛡️ 誠実な設計

法令・金額・可否を断定しない。住所は監査ログでマスク。出典をでっち上げない。全ローカルで PII は手元に。

なぜ無料で公開するのか

同じ概念が会社・ポータル・担当者ごとに違う言葉で書かれ、標準の共通語彙が公共財として存在しない。 AI・OCR・データ連携のたびに各社が名寄せ辞書を作り直している。共通語彙は競争領域ではない。だから 1 つのオープンな辞書にする。
